# 📖 Project Story

Efficient supply chain management is one of the most important factors for a company's success. Managing suppliers, inventory, shipments, and customer deliveries requires accurate data and meaningful insights.

To address this, I built an interactive **Samsung Supply Chain & Logistics Dashboard** in **Power BI**. This dashboard enables businesses to monitor inventory levels, supplier performance, shipment status, customer revenue, and logistics operations—all in one place.

The goal of this project is to transform raw operational data into clear, interactive dashboards that support smarter and faster business decisions.

# 🎯 Project Objectives

This dashboard was created to answer important business questions such as:

✅ How much inventory is currently available?

✅ Which suppliers perform the best?

✅ Which products have the highest defect rate?

✅ What is the shipment delivery performance?

✅ Which customers generate the highest revenue?

✅ Which products require inventory replenishment?
